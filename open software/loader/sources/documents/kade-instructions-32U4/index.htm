Programming Instructions

  1. Connect your KADE miniArcade device to a USB port.

  2. If you have not previously programmed KADE then you can skip to step 3 - it should be in program mode already.
    Otherwise, you will need to press the small buttons in the following sequence while KADE is connected to a USB port:

    • Press and hold the RESET button
    • Press and hold the HWB button
    • Release the RESET button
    • Release the HWB button

    This can be done by rolling your finger over the buttons as per the animation on the right.

  3. Select a firmware from the list of those available.

  4. Click the “Program KADE with Selected Firmware” button.
    The firmware should be updated in a few seconds.

  5. Unplug device. It is now ready to be used as an encoder.

For further information please refer to the quick start guide and the kadevice.com web site.
